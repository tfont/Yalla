- [mb_detect_encoding()](mb_detect_encoding.php) (4.0.6+)
- [money_format()](money_format.php) (WIN OS)
- [str_split()](str_split.php) (5.0+)
- [trait_exists()](trait_exists.php) (5.4.0+)
- [utf8_encode()](utf8_encode.php) (4.0+)
